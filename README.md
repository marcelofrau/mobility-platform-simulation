# mobility-platform-simulation

## How to run frontend

```shell
cd mps-frontend
npm install
npm start
```

You can access the frontend on the url: http://localhost:3000/

## How to run backend

**Note**: To run backend you must have a mongodb initialized on localhost and default port.

```shell
cd mps-backend
npm install
npm start
```
